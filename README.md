### Project README

# Sweet Kneads Bakery Landing Page

A modern, high-performance landing page designed for a professional bakery. This project focuses on clean UI/UX, mobile responsiveness, and high-quality visual storytelling to drive customer engagement.

### Key Features

* Responsive Design: Seamlessly transitions between mobile, tablet, and desktop views.
* Interactive Hero: Features a floating transparent image and an embedded promotional video.
* Dynamic Sections: Includes a categorized menu, a weekend discount grid with hover effects, and a photo gallery.
* Lead Generation: Functional contact form with validation and a direct WhatsApp integration button.
* Performance: Built with vanilla technologies to ensure fast loading times.

### Tech Stack

* HTML5: Semantic structure for SEO and accessibility.
* CSS3: Custom properties (variables), Flexbox, and CSS Grid for layout.
* JavaScript: Vanilla ES6 for menu toggles and smooth scroll interactions.
* Font Awesome: High-quality vector icons for social media and UI elements.

### How to Use

1. Clone the repository.
2. Open index.html in any modern web browser.
3. No build tools or dependencies required.
